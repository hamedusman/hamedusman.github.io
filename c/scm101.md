The standardized logic to calculate your code is as follows:
<ul style="list-style-type:square;">
   <li>Step 1. Student name - Use only first and last names, even your name has one or more than one middle name(s).</li>
    <ul style="list-style-type:none;">
     <ol>
      <li>Take the last three letters of your first and last names.</li> 
      <li>Assign each letter a number (e.g., A = 1; B = 2; C = 3 ... X = 24; Y = 25; Z = 26).</li>
       <ul style="list-style-type:none;"><li>First name (HAMED): M = 13; E = 5; D = 4) and last name (USMAN): M = 13; A = 1; N = 14</li></ul>
      <li>Sum of last 3 letters of the first name = M(13) + E(5) + D(4) = 22 </li>
      <li>Sum of last 3 letters of the last name = M(13) + A(1) + N(14) = 28 </li>
     </ol>
    </ul>
   <li>Step 2. Registration number: 123456 and enrollment number: 12-345678-910 - The length of these numbers may vary.</li>
    <ul style="list-style-type:none;">
     <ol>
      <li>Take the last three digits of both registration (456) and enrollment (910) numbers.</li>
      <li>Join these digits (456 and 910) together to form a new number = 456910 </li>
     </ol>
    </ul>
   <li>Step 3. Final addition: 22 + 28 + 456910 = 456960 - This is your final student code.</li>
</ul>

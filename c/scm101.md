Follow these steps to calculate your unique student code (important to know your scores):
<ul style="list-style-type:square;">
   <li>Step 1: Use only first and last names, even your name has one or more than one middle name(s)</li>
    <ul style="list-style-type:none;">
     <ol>
      <li>Take the last three letters of both first and last names (example: HAMED USMAN)</li> 
      <li>Assign these letters a number (e.g., A = 1, B = 2, C = 3, ... X = 24, Y = 25, Z = 26)</li>
      <li>Add the assigned values of last 3 letters of first name (HAMED) = M(13) + E(5) + D(4) = 22 </li>
      <li>Add the assigned values of last 3 letters of last name (USMAN) = M(13) + A(1) + N(14) = 28 </li>
     </ol>
    </ul>
   <li>Step 2: Use your own registration (e.g., 123456) and enrollment (e.g., 12-345678-910) numbers of the university</li>
    <ul style="list-style-type:none;">
     <ol>
      <li>Take the last three digits of both registration (456) and enrollment (910) numbers</li>
      <li>Join these digits (456 and 910) together to form a new number = 456910 </li>
     </ol>
    </ul>
   <li>Step 3: Finally add all numbers: 22 + 28 + 456910 = 456960 - This is your final student code!</li>
</ul>
